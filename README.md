# blind_watermark-

基于浏览器的图片盲水印加密/解密工具，所有处理在本地完成，无需上传。

## 功能特性

- **嵌入水印**：将文本信息以不可见水印的形式嵌入图片，肉眼无法察觉
- **提取水印**：从已嵌入水印的图片中提取原始文本信息
- **密码保护**：使用密码对水印进行加密，未提供正确密码无法提取
- **完全本地**：所有图像处理均在浏览器 JavaScript 引擎中完成，不会向任何服务器发送数据
- **参数可调**：支持调节鲁棒性强度（d1）和双重嵌入强度（d2）

## 算法原理

本项目采用与 [blind_watermark](https://github.com/guofei9987/blind_watermark) 相同的 **DWT-DCT-SVD** 混合频域算法：

1. **Haar 小波变换 (DWT)**：将图像从空间域转换到频域，分离低频和高频分量
2. **离散余弦变换 (DCT)**：对低频分量进行 4x4 分块 DCT 变换
3. **奇异值分解 (SVD)**：对每个 DCT 块进行 SVD 分解，通过量化奇异值嵌入水印位
4. **密码置乱**：使用密码驱动的随机排列对水印位序和 DCT 系数进行双重加密

该算法具有良好的不可见性和抗 JPEG 压缩能力。

## 使用方法

直接在浏览器中打开 `index.html` 即可使用，无需安装任何依赖。

### 嵌入水印

1. 上传原始图片（支持 JPG / PNG / WebP）
2. 输入水印文本内容和密码
3. 根据需要调节鲁棒性参数
4. 点击「嵌入水印」
5. 预览对比后下载水印图
6. **记住显示的字节数**，提取时需要用到

### 提取水印

1. 上传带有水印的图片
2. 输入嵌入时使用的相同密码
3. 输入嵌入时显示的 UTF-8 字节数
4. 根据需要调节鲁棒性参数（需与嵌入时一致）
5. 点击「提取水印」

## 技术实现

- **纯前端**：使用原生 JavaScript 实现全部算法，无第三方库依赖
- **数值计算**：JavaScript 实现的 Haar DWT、4x4 DCT/IDCT、Golub-Kahan SVD 分解
- **主题切换**：支持浅色、深色、暖灰三种外观主题
- **响应式布局**：适配桌面和移动端浏览器
